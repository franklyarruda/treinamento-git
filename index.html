Quais os comandos para configurar usuário e email no git??

1ª resposta: git config --global user.name "w3schools-test" git config --global user.email "test@w3schools.com"

Qual a diferença entre git e github?

2ª resposta: git é um software VCS local que permite aos desenvolvedores
salvar snapshots de seus projetos ao longo do tempo. Geralmente
é melhor para uso individual. GitHub é uma plataforma baseada
na web que incorpora os recursos de controle de versões do gitHub
para que possam ser usados colaborativamente.

O que faz o comando commit?

3º resposta: O comando git commit captura um instantâneo das mudanças
preparadas do projeto no momento.

O que faz o comando push?

4º resposta: O comando push transfere commits do repositório local a um
repositório remoto.
